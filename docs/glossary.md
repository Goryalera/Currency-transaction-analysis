# Glossary

| Термин | Значение |
|---|---|
| FX Spot | Валютная сделка с расчетами обычно на T+2 |
| Currency pair | Валютная пара, например EUR/USD |
| Buy/Sell | Направление сделки |
| Notional | Номинальная сумма сделки |
| Exchange rate | Курс сделки или рыночный курс |
| Trade date | Дата заключения сделки |
| Value date | Дата расчетов |
| Quote | Котировка, предложенная клиенту |
| Counterparty | Контрагент сделки |
| Limit check | Проверка доступного торгового лимита |
| Confirmation | Подтверждение условий сделки |
| SSI | Standard Settlement Instructions, расчетные реквизиты |
| Reconciliation | Сверка данных после проведения расчетов |
| STP | Straight-through processing, прохождение без ручного вмешательства |

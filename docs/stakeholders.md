# Stakeholders

| Участник | Ответственность |
|---|---|
| Корпоративный клиент | Отправляет запрос и подтверждает условия |
| Sales | Получает запрос, уточняет данные и взаимодействует с клиентом |
| Trader | Формирует котировку |
| Risk/Compliance | Проверяет клиента, лимиты и ограничения |
| Middle Office | Контролирует корректность зарегистрированной сделки |
| Operations/Back Office | Организует подтверждение, расчеты и сверку |
| Информационная система | Хранит данные, выполняет автоматические проверки и маршрутизацию |
